Brief Description

For our final project, we added surface dynamics to our mesh editor from assignment 2. Leveraging symplectic and forward Eulerian / discretization methods, and the powerful Laplace-beltrami operator, we added dynamic simulations for the laplace equation, heat equation, wave equation, and mean curvature flow.

Building and running code

  1. Run the following command from the root of the project:
    mkdir build && cd build && cmake .. && make
  2. From here on you can run the application as you would project 2
    ./meshedit ../dae/collada_file.dae

Once you've loaded the code, you use the following commands to interact with the mesh (first letter is the command -- select a vertex for commands 2 - 8):

  • 1 - Laplace Equation with Symplectic Euler
  • 2 - Laplace Equation with Backward Euler
  • 3 - Heat Equation with Symplectic Euler
  • 4 - Heat Equation with Backward Euler
  • 5 - Wave Equation with Symplectic Euler
  • 6 - Wave Equation with Backward Euler
  • 7 - Mean Curvature Flow with Symplectic Euler
  • 8 - Mean Curvature Flow with Backward Euler
  • [ - Make timestep smaller
  • ] - Make timestep larger
  • a or A - Choose random vertex on mesh and give it a displacement of 3
