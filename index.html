Brief Description

We are going to add surface dynamics to our mesh editor from assignment 2. This will enable us to render realistic 3D animations such as waves etc. We will start off by implementing the Laplace, Heat and Wave equations to simulate the appropriate dynamics, updating vertex positions using the symplectic Euler method. We will then move on to explore more advanced solvers, and possibly implement different PDEs to simulate different dynamics.

Implementation Steps

  1. We will first implement the three basic linear equations: the Laplacian, the heat equation, and the wave equation.
  2. We will then implement some more advanced solvers. The current plan is to implement the heat equation using backwards Euler, but we may do other things too after reading more carefully through the provided notes.
  3. If we have time (or decide to only do one advanced solver), we'll implement some more advanced equations as well. The Bubble Dynamics and Mesh Smoothing look particularly interesting.

NOTE: We do not plan on handling boundary conditions in our implementation.
