Brief Description

We are going to implement the kPhone 869 Camera Pipeline project to learn about image processing. In the first part of the project, we will write code to process the raw output of a camera and filter the image output to remove artifacts. In the second part of the project we will implement auto-focusing functionality for a camera. We will attempt to implement a fast algorithm to reflect the fact that quick auto-focusing is necessary in real-world usage to capture quick shots.

Implementation Steps

  1. We will first implement the three basic linear equations: the Laplacian, the heat equation, and the wave equation.
  2. We will then implement some more advanced solvers. The current plan is to implement the heat equation using backwards Euler, but we may do other things too after reading more carefully through the provided notes.
  3. If we have time (or decide to only do one advanced solver), we'll implement some more advanced equations as well. The Bubble Dynamics and Mesh Smoothing look particularly interesting.
